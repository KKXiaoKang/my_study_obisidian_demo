检查当前分支情况，然后清理远程已删除的本地分支。

**总结一下清理步骤：**

1. **更新远程引用并清理已删除的远程分支：**
   ```bash
   git fetch --prune
   ```

2. **找出远程已不存在的本地分支：**
   ```bash
   git branch | sed 's/^[ *]*//' | while read branch; do 
     if [ "$branch" != "main" ] && ! git ls-remote --heads origin "$branch" | grep -q .; then 
       echo "$branch"; 
     fi; 
   done
   ```

3. **删除这些分支（强制删除未合并的分支）：**
   ```bash
   git branch -D <分支名>
   ```

**或者使用一条命令自动清理所有远程已不存在的本地分支：**
```bash
git fetch --prune && git branch | sed 's/^[ *]*//' | while read branch; do 
  if [ "$branch" != "main" ] && ! git ls-remote --heads origin "$branch" | grep -q .; then 
    git branch -D "$branch"; 
  fi; 
done
```
